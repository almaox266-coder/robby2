ทีเด็ดบอลวันนี้ & การเดิมพันฟุตบอล: เรทราคาบอลสด

สรุปสั้น: ส่วนขยายนี้ไม่เก็บรวบรวม ไม่ส่ง และไม่แชร์ข้อมูลส่วนบุคคลใดๆ ของผู้ใช้ ข้อมูลทั้งหมดจะถูกเก็บไว้ในเครื่องของคุณเท่านั้น

1. บทนำ

นโยบายความเป็นส่วนตัวฉบับนี้อธิบายว่าส่วนขยาย Chrome "ทีเด็ดบอลวันนี้ & การเดิมพันฟุตบอล: เรทราคาบอลสด" (ต่อไปนี้เรียกว่า "ส่วนขยาย") จัดการข้อมูลของผู้ใช้อย่างไร เราให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้เป็นอย่างยิ่ง

2. ข้อมูลที่เราเก็บรวบรวม

ส่วนขยายนี้ ไม่เก็บรวบรวมข้อมูลส่วนบุคคลใดๆ เราไม่เก็บ:

  • ชื่อ อีเมล หมายเลขโทรศัพท์ หรือข้อมูลระบุตัวตนอื่นๆ
  • ประวัติการท่องเว็บ
  • ข้อมูลตำแหน่งที่ตั้ง
  • ข้อมูลทางการเงิน
  • ข้อมูลการสื่อสาร

3. ข้อมูลที่จัดเก็บในเครื่อง

ส่วนขยายจัดเก็บข้อมูลต่อไปนี้ในเครื่องของผู้ใช้ผ่าน chrome.storage.local เท่านั้น:

ประเภทข้อมูลวัตถุประสงค์ที่จัดเก็บ
การตั้งค่าธีม (มืด/สว่าง)จดจำธีมที่ผู้ใช้เลือกเครื่องของผู้ใช้เท่านั้น
รายการทีมโปรดแสดงแมตช์ของทีมที่ผู้ใช้ติดตามเครื่องของผู้ใช้เท่านั้น
การตั้งค่าการแจ้งเตือนส่งการแจ้งเตือนตามที่ผู้ใช้กำหนดเครื่องของผู้ใช้เท่านั้น
การตั้งค่าลีกจดจำลีกที่ผู้ใช้เลือกดูเครื่องของผู้ใช้เท่านั้น
แคชข้อมูลแมตช์แสดงผลเร็วขึ้น ลดการเรียก APIเครื่องของผู้ใช้เท่านั้น

ข้อมูลเหล่านี้ ไม่ถูกส่งไปยังเซิร์ฟเวอร์ภายนอก และผู้ใช้สามารถลบได้ตลอดเวลาโดยถอนการติดตั้งส่วนขยาย

4. การเชื่อมต่อเครือข่าย

ส่วนขยายเชื่อมต่อกับบริการภายนอกเพียงแห่งเดียว:

  • API-Sports Football API (https://v3.football.api-sports.io) — เพื่อดึงข้อมูลผลบอลสด ตารางแข่งขัน ราคาบอล และสถิติทีม

การเชื่อมต่อนี้ส่งเฉพาะคำขอข้อมูลฟุตบอล (เช่น รหัสลีก, วันที่แข่งขัน) ไม่มีข้อมูลส่วนบุคคลของผู้ใช้ถูกส่งไป

5. สิทธิ์ที่ขอใช้

สิทธิ์เหตุผล
storageจัดเก็บการตั้งค่าของผู้ใช้ในเครื่อง (ธีม, ทีมโปรด, การแจ้งเตือน)
alarmsตั้งเวลาอัปเดตผลบอลสดอัตโนมัติทุก 5 นาที
notificationsแจ้งเตือนเมื่อทีมโปรดยิงประตูหรือแมตช์เริ่ม
Host permissionเข้าถึง API-Sports เพื่อดึงข้อมูลฟุตบอลสด

6. คุกกี้และการติดตาม

ส่วนขยายนี้:

  • ไม่ใช้คุกกี้
  • ไม่ใช้เครื่องมือวิเคราะห์ (analytics) ใดๆ
  • ไม่ติดตามพฤติกรรมผู้ใช้
  • ไม่แสดงโฆษณา
  • ไม่มีโค้ดของบุคคลที่สาม (third-party tracking)

7. การแชร์ข้อมูล

เรา ไม่แชร์ ไม่ขาย ไม่แลกเปลี่ยน ข้อมูลใดๆ ของผู้ใช้กับบุคคลที่สาม

8. ความปลอดภัยของข้อมูล

ข้อมูลทั้งหมดถูกจัดเก็บในเครื่องของผู้ใช้ผ่าน API ของ Chrome ที่มีระบบรักษาความปลอดภัยในตัว ไม่มีข้อมูลถูกส่งผ่านเครือข่ายยกเว้นคำขอข้อมูลฟุตบอลไปยัง API-Sports

9. สิทธิ์ของผู้ใช้

ผู้ใช้สามารถ:

  • ดูข้อมูลที่จัดเก็บ — ผ่านหน้าการตั้งค่าของส่วนขยาย
  • ลบข้อมูลทั้งหมด — โดยถอนการติดตั้งส่วนขยาย หรือล้างข้อมูลส่วนขยายผ่านการตั้งค่า Chrome
  • ปิดการแจ้งเตือน — ผ่านหน้าการตั้งค่าของส่วนขยาย
  • ปิดวิดเจ็ตลอยตัว — ผ่านหน้าการตั้งค่าของส่วนขยาย

10. เด็กและเยาวชน

ส่วนขยายนี้ไม่ได้ออกแบบมาสำหรับเด็กอายุต่ำกว่า 13 ปี และไม่เก็บรวบรวมข้อมูลจากเด็กโดยเจตนา

11. การเปลี่ยนแปลงนโยบาย

หากมีการเปลี่ยนแปลงนโยบายความเป็นส่วนตัว เราจะอัปเดตหน้านี้พร้อมระบุวันที่แก้ไข ขอแนะนำให้ตรวจสอบหน้านี้เป็นระยะ

12. ติดต่อเรา

หากมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัว สามารถติดต่อผ่าน:

  • หน้า Support ของ Chrome Web Store
Summary: This extension does not collect, transmit, or share any personal data. All information is stored locally on your device only.

1. Introduction

This Privacy Policy explains how the Chrome extension "ทีเด็ดบอลวันนี้ & การเดิมพันฟุตบอล: เรทราคาบอลสด" (Football Tips Today & Betting: Live Odds) (hereinafter "the Extension") handles user data. We take user privacy very seriously.

2. Information We Collect

This extension does not collect any personal information. We do not collect:

  • Names, emails, phone numbers, or other identifying information
  • Browsing history
  • Location data
  • Financial information
  • Communication data

3. Locally Stored Data

The extension stores the following data locally on the user's device via chrome.storage.local only:

Data TypePurposeStorage Location
Theme preference (dark/light)Remember user's theme choiceUser's device only
Favorite teams listShow matches for followed teamsUser's device only
Notification settingsSend notifications as configuredUser's device only
League preferencesRemember selected leaguesUser's device only
Cached match dataFaster rendering, reduce API callsUser's device only

This data is never transmitted to any external server and can be deleted at any time by uninstalling the extension.

4. Network Connections

The extension connects to only one external service:

  • API-Sports Football API (https://v3.football.api-sports.io) — to fetch live football scores, fixtures, odds, and team statistics.

These requests only send football data queries (e.g., league IDs, match dates). No personal user data is transmitted.

5. Permissions Used

PermissionReason
storageStore user preferences locally (theme, favorites, notifications)
alarmsSchedule automatic live score updates every 5 minutes
notificationsAlert when favorite teams score or matches begin
Host permissionAccess API-Sports to fetch live football data

6. Cookies and Tracking

This extension:

  • Does NOT use cookies
  • Does NOT use any analytics tools
  • Does NOT track user behavior
  • Does NOT display advertisements
  • Does NOT contain third-party tracking code

7. Data Sharing

We do not share, sell, or trade any user data with third parties.

8. Data Security

All data is stored locally on the user's device through Chrome's built-in secure storage APIs. No data is transmitted over the network except football data requests to API-Sports.

9. User Rights

Users can:

  • View stored data — through the extension's settings page
  • Delete all data — by uninstalling the extension or clearing extension data in Chrome settings
  • Disable notifications — through the extension's settings page
  • Disable floating widget — through the extension's settings page

10. Children's Privacy

This extension is not designed for children under 13 and does not knowingly collect data from children.

11. Changes to This Policy

If this privacy policy is updated, we will update this page with the revision date. We recommend checking this page periodically.

12. Contact Us

If you have questions about this privacy policy, please contact us through:
